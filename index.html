Landing rockets like a boss

FALL is a Fully Autonomous Launch and Landing module for KOS mod. This personal project has been refined and documented for educational and practical purposes. If you enjoy fun experiences with less explosions look no further, the magic is about to happen.

Get started now View it on GitHub

Unedited footage without distractions can also be found here.

Dependencies

Currently the module requires these two mods:

  • Kerbal Operating System
  • Trajectories

Installation

Download the zip file or clone repo and extract its contents to your archive directory path:

$ ../KerbalSpaceProgram/Ships/Script/

Done!
